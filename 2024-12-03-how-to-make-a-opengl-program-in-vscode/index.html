一、关于 g++ 的配置问题

我使用的是 msys64 下面的 mingw64。

假如 msys64 的软件根目录为 D:\Software\msys64\，那么我使用的 g++.exe 位于 D:\Software\msys64\mingw64\bin 下面。

为了让 VS Code 找到 g++.exe，需要配置 Path 环境变量。在 Windows 的环境变量设置里面，加入 D:\Software\msys64\mingw64\bin 即可。
